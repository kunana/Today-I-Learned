# Trigonometric_Funtion

> ### 간단한 삼각함수 정리

`삼각함수는 직각 삼각형에 대해서 하나의 각이 주어졌을 때 각 변에 대한 비율을 구해준다.`


$sin = 빗변과 높이의 비율$

$cos = 빗변과 밑변의 비율$

$tan = 밑변과 높이의 비율$
